Signature Analysis
Release Date: September 2025
Signature Analysis is an AI model capable of recognizing my handwriting. It was trained on around 4000 samples of the letter “A” written by me (labeled 1) and others (labeled 0). I decided to base the project only on the letter “A” to reduce the sample space and the required training data, as it would reduce variance. Visit the GitHub page linked below the title for more information.
